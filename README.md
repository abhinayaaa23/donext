# DoNext

## Plan Less. Execute More.

DoNext is an AI-powered productivity companion designed to help users move from intention to execution.

Unlike traditional task managers that simply store tasks and send reminders, DoNext helps users understand workload, estimate effort, break down complex goals, prioritize effectively, and stay ahead of deadlines through AI-powered planning assistance.

## Problem Statement

### The Last-Minute Life Saver

Students, professionals, researchers, and entrepreneurs often struggle to manage multiple deadlines and commitments simultaneously.

Most productivity tools are reactive—they remind users about tasks but provide little assistance in understanding workload, prioritizing effectively, or planning realistically.

DoNext was built to bridge the gap between planning and execution.

---

## Key Features

- **AI-Powered Effort Estimation** – Uses Gemini to estimate the time and effort required for tasks.

- **Task Decomposition** – Breaks complex goals into smaller, actionable subtasks.

- **Workload Forecasting** – Predicts future workload distribution to help users avoid last-minute rushes.

- **Smart Prioritization** – Helps users identify the most important tasks based on urgency, effort, and deadlines.

- **Nudge AI Assistant** – Provides contextual productivity guidance and planning recommendations.

- **Focus Sessions** – Supports deep work through structured focus tracking.

- **Productivity Insights** – Generates analytics and behavioral insights based on user activity.

- **Secure Authentication** – Google Sign-In powered by Firebase Authentication.

- **Cloud-Synced Data** – Task and productivity data stored securely using Cloud Firestore.
